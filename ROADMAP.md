# OrbitGraph roadmap

## Current development: v1.1.0

- [x] Desktop, touch, and keyboard camera navigation
- [x] Responsive mobile camera controls
- [x] Keyboard accessibility
- [x] PNG and JSON exports
- [x] React ref API for interactions and exports
- [x] Test coverage and documentation for v1.1 APIs

## Planned: v1.2.0 — scalable graph intelligence

- [ ] Move force simulation to a Web Worker
- [ ] Run expensive graph analysis outside the main UI thread
- [ ] Connected-component and community detection
- [ ] Centrality metrics such as degree, weighted degree, and PageRank
- [ ] Optional visual highlighting for communities and important nodes
- [ ] Analysis benchmarks for large graphs

## Future

- [ ] Additional renderer adapters and framework integrations
- [ ] Advanced theming and styling hooks
- [ ] AR/VR exploration experiments
- [ ] Server-side graph transformation utilities

The roadmap describes intent, not a release contract. Feedback and contributions are welcome through GitHub issues and discussions.
